# ABAP RAP Code Examples

This folder contains all ABAP RAP code examples. Below is a list of the files included:

- `ZTRAVEL_XXX.txt` for database table
- `ZI_TRAVEL_XXX.ddls` for Interface CDS View
- `ZC_TRAVEL_XXX.ddls` for Consumption CDS View
- `ZI_TRAVEL_XXX.bdef` for Interface Behavior Definition
- `ZC_TRAVEL_XXX.bdef` for Consumption Behavior Definition
- `ZBP_I_TRAVEL_XXX.clas.abap` for global behavior implementation
- `ZBP_I_TRAVEL_XXX.clas.locals_imp.abap` for local implementation
- `Z_TRAVEL_XXX.srvd` for Service Definition

## Usage Instructions

To create in order from database table through service binding, please follow the steps outlined in the respective files. Note that you should replace `XXX` with your user initials.
